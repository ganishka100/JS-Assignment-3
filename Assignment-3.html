Explain the Assignment Operator. Ans3. The assignment operator in JavaScript is denoted by the equals sign (=). It is used to assign a value to a variable. The syntax is as follows: variable = expression; variable is the name of the variable, and expression is the value or calculation that you want to assign to the variable. let x = 10; // Assigning the value 10 to the variable x let y = x + 5; // Assigning the result of the expression x + 5 to the variable y ------------------------------------------------------------------------------------------- Q4. In JavaScript, both pre-increment and post-increment are unary operators that increase the value of a variable by 1. 1. Pre-Increment (++variable): In pre-increment, the increment operation is performed before the value of the variable is used in the expression. let x = 5; let y = ++x; console.log(x); // Output: 6 console.log(y); // Output: 6 2. Post-Increment (variable++): In post-increment, the increment operation is performed after the current value of the variable is used in the expression. let a = 5; let b = a++; console.log(a); // Output: 6 console.log(b); // Output: 5
